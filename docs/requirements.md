# Your Weather — 需求文档

## 1. 项目概述

一个个人使用的天气查询 Web 应用,提供城市天气查询、5 天预报、地理定位、城市收藏功能。中文界面,聚焦轻量、好看、好用。

## 2. 目标用户与场景

- **主要用户**:个人/中文用户,需要快速查询某个城市天气
- **典型场景**:
  - 打开应用,自动定位显示当前位置天气
  - 搜索某个城市看天气
  - 把常去的城市收藏起来,下次直接点击查询
  - 查看未来 5 天预报安排出行

## 3. 功能需求

### 3.1 当前天气查询(P0)
- **输入**:城市名(中文/英文/拼音)
- **输出**:
  - 城市名 + 国家/省份
  - 当前温度(整数)
  - 体感温度
  - 天气状况文本(如"小雨")+ 天气图标
  - 湿度(%)
  - 风速(可切换 m/s / km/h)+ 风向
  - 气压(hPa)
  - 能见度(km)
  - 日出/日落时间
  - 数据更新时间

### 3.2 5 天预报(P0)
- 每天显示:日期、白天/夜间天气图标、最高/最低温度、降水概率
- 横向滚动或网格展示

### 3.3 城市搜索(P0)
- 输入框防抖(300ms),调用 Geocoding API 显示候选项
- 候选项显示城市名、省份、国家
- 点击候选项执行查询
- 支持回车直接搜索第一个候选项
- 搜索失败提示"未找到该城市"

### 3.4 地理定位(P0)
- 首次进入应用询问定位权限
- 用户授权后调用 `navigator.geolocation`,反向地理编码得到城市名
- 拒绝授权:不报错,显示空状态引导搜索或选收藏
- 不可用(浏览器不支持/HTTPS 不满足):同上

### 3.5 城市收藏(P0)
- 当前查询页面有"收藏"按钮,点击加入收藏
- 收藏列表显示在主页侧栏/下方,显示城市名
- 点击收藏项 → 切换查询该城市
- 长按/右键/删除按钮 → 移除收藏
- 持久化:LocalStorage,key = `your-weather:favorites`,value = `{ id, name, country, lat, lon }[]`
- 上限:20 个(超出时提示)

### 3.6 单位切换(P0)
- 顶部切换:温度 °C / °F,风速 m/s / km/h
- 持久化到 LocalStorage(`your-weather:units`)

### 3.7 动态背景(P1)
- 根据当前天气状况切换背景:
  - 晴天:暖蓝渐变 + 太阳光晕
  - 多云:灰蓝 + 云层
  - 雨/雪:深色 + 雨滴/雪花动效(纯 CSS)
  - 夜间(根据日出日落判断):深紫/深蓝
- 不能影响可读性

## 4. 非功能需求

### 4.1 性能
- 首屏 < 2s(本地 Vite dev 环境)
- API 响应缓存 10 分钟(同城市+同语言)
- 路由级 lazy load 暂不需要(单页)

### 4.2 可用性
- 移动端(375px+)和桌面端(1024px+)自适应
- 所有 API 调用有 loading 骨架
- 所有失败状态有清晰文案

### 4.3 错误处理
| 场景 | 处理 |
|---|---|
| API key 缺失 | 启动时控制台报错 + 顶部 banner 提示 |
| 网络断连 | "网络异常,请检查后重试" + 重试按钮 |
| 429 限流 | "请求过于频繁,稍后再试" |
| 城市未找到 | 搜索框下方提示 |
| 定位被拒 | 静默,显示引导文案 |

### 4.4 浏览器兼容
- 现代浏览器(Chrome/Safari/Edge/Firefox 最近两年版本)
- 不考虑 IE

### 4.5 隐私
- 不收集任何用户数据
- 定位信息仅本地使用,不发送给第三方
- API key 在前端是暴露的(OpenWeatherMap 免费 key 限制 60 次/分钟,接受这个风险)

## 5. 范围之外(明确不做)

- ❌ 小时级预报(本期)
- ❌ AQI 空气质量
- ❌ 收藏列表预览温度
- ❌ 收藏排序拖拽
- ❌ 多语言(仅中文)
- ❌ 深色/浅色模式切换(用动态背景代替)
- ❌ PWA / 离线
- ❌ 后端服务(纯前端)
- ❌ 用户账号

## 6. 验收标准

- [ ] 输入"上海"能查到上海天气
- [ ] 输入英文"Tokyo"也能查到
- [ ] 拼错的城市有明确错误提示
- [ ] 首次打开有定位询问,授权后自动显示本地天气
- [ ] 拒绝定位不报错
- [ ] 5 天预报数据完整
- [ ] 收藏/取消收藏正常工作,刷新后保留
- [ ] °C ↔ °F 切换全页面同步
- [ ] 晴天/雨天背景视觉有明显区别
- [ ] 移动端布局不溢出
